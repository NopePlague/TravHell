# TravHell — Discover Places Only Locals Know About
<h3>What's this about?</h3>

You know that feeling when you travel somewhere and you only end up visiting the same spots everyone else does — just because they're on Google Maps? Meanwhile, locals know about this amazing viewpoint, a tiny dhaba that serves the best food, or a hidden waterfall that barely anyone visits.
That's exactly the problem this app tries to solve.
<br>
TravHell is a web app that helps you discover offbeat, lesser-known spots at any destination — the kind of places you'd normally only find out about from a local or stumble across on someone's reel. Think hidden photo spots, budget-friendly dhabas, scenic viewpoints, and more.

<br>
<br>

<h3>What the app does:</h3>


<pre>Search for any city or destination</pre>
<pre>Browse lesser-known and interesting places around that area</pre>
<pre>Filter spots by category — food, nature, photo spots, hidden gems, etc.</pre>
<pre>Sort results by rating or relevance</pre>
<pre>Save your favorite spots so you don't lose track of them</pre>
<pre>Works on mobile, tablet, and desktop</pre>

<br>
<br>

<h3>Tech Stack:</h3>
<pre>HTML, CSS, JavaScript — the core of everything</pre>
<pre>Fetch API — to pull live data from the places API</pre>
<pre>OpenTripMap API — provides data on attractions and points of interest worldwide, including lesser-known spots</pre>
<pre>Array Higher-Order Functions — used for all search, filter, and sort logic (.filter(), .map(), .sort(), etc.)</pre>
<pre>Local Storage — to save your favorite spots between sessions</pre>
<pre>Optionally: Tailwind CSS for styling</pre>

<br>
<br>

<h3>Planned Features:</h3>
<pre>Search — find spots by city or keyword</pre>
<pre>Filter by category — nature, food, photography, hidden gems</pre>
<pre>Sort — by rating, name, or distance</pre>
<pre>Save Favorites — bookmark spots you want to visit, stored in local storage</pre>
<pre>Dark / Light Mode — theme toggle for better experience</pre>
<pre>Loading indicators — visual feedback while data is being fetched</pre>
<pre>Pagination — browse results page by page without overwhelming the screen</pre>

<br>
<br>

<h3>API Being Used:</h3>


<pre>Geoapify =></pre>

Website: https://www.geoapify.com/places-api/

Docs: https://apidocs.geoapify.com/docs/places/

Why this one? It has a large database of places including offbeat, cultural, and historic spots that don't always show up on mainstream travel apps. It covers locations worldwide, supports category-based filtering, and has a free tier which is perfect for this project.

<br>
<br>

<h3>How to Run This Project:</h3>

No complicated setup needed — it's a plain HTML/CSS/JS project.

<pre>Clone the repo: git clone https://github.com/NopePlague/TravHell.git</pre>

Open the project folder and just open index.html in your browser — that's it.
If you want to use the live API, get a free API key from https://opentripmap.io and add it to the config section in app.js.
